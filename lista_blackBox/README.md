# Lista BlackBox - 10 Exercícios de Lógica com Listas/Arrays

Criados baseados no estilo das listas existentes (prompts, output em #resultado).

Abra cada exN.html no navegador, complete o JS.

**Exercícios:**
1. Soma de 5 elementos.
2. Média aritmética.
3. Maior e menor.
4. Contar pares.
5. Inverter lista.
6. Buscar posição de número.
7. Filtrar positivos.
8. Ordenar crescente.
9. Matriz 3x3, soma linhas.
10. Frequência, mais comum.

Teste com `start http://localhost` ou abra arquivos diretamente.
